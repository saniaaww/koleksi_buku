# 📚 Koleksi Buku

Koleksi Buku adalah aplikasi berbasis web untuk mengelola, menampilkan, sekaligus melakukan penjualan buku.  
Project ini dikembangkan sebagai bagian dari pembelajaran mata kuliah **Workshop Pengembangan Perangkat Lunak Web (Framework) Praktikum Semester 4** dan juga portofolio pribadi.

---

## 🔑 Fitur
- **Manajemen Koleksi**: tambah, edit, dan hapus data buku.  
- **Pencarian Cepat**: cari buku berdasarkan judul, penulis, atau kategori.  
- **Penjualan & Kasir**: modul kasir untuk transaksi langsung.  
- **Geolocation Sales**: membantu sales menemukan toko dalam radius tertentu.  
- **Tag Harga & Barcode**: cetak harga dan barcode ke kertas tag.  
- **Scan Buku**: pilih buku dengan sistem scan.  
- **UI Responsif**: menggunakan Bootstrap template.  
- **Database MySQL**: integrasi penuh dengan PHP.  
- **Teknologi Modern**: HTML + DataTables, AJAX, dan Axios.

---

## ⚙️ Teknologi
- **Backend**: PHP  
- **Frontend**: Bootstrap, DataTables  
- **Database**: MySQL  
- **Integrasi**: AJAX & Axios
